# Technology Stack

## Frontend (react-frontend)
- **Framework**: React 19.2.4 (Create React App template)
- **State Management**: React `useState` / `useEffect` (Manual State)
- **API Client**: Axios 1.13.4
- **Styling**: Vanilla CSS3 with Modern Variables
- **Testing**: Jest, React Testing Library
- **Build Tool**: `react-scripts` 5.0.1

## Backend (strapi-backend)
- **Framework**: Strapi 5.34.0 (Headless CMS)
- **Database**: SQLite (via `better-sqlite3` 12.4.1)
- **ORM/Query Builder**: Strapi Internal (Knex-based)
- **Language**: JavaScript / TypeScript (Ready)
- **Runtime**: Node.js (>=20.0.0)

## Infrastructure
- **Environment**: Development
- **Ports**: 
  - Frontend: 3000
  - Backend: 1337
